# FitFlow 비즈니스 규칙

> 이 문서는 FitFlow 시스템의 확정된 비즈니스 규칙을 정의합니다.  
> PRD 또는 다른 문서와 상충 시 이 문서가 우선합니다.

---

## BR-1. 회원 상태 관리

| 상태 | 설명 | 전환 조건 |
|------|------|-----------|
| active | 유효한 회원권 보유 | 회원권 활성 상태 |
| inactive | 회원권 만료 또는 미보유 | 회원권 만료 후 자동 전환 |
| frozen | 회원권 동결 중 | 동결 신청 시 |

- 회원 탈퇴는 소프트 삭제(`deleted_at` 필드)로 처리
- 센터 이관 시 `transfer_history`에 기록하며, 이관 후에는 **새 센터에서만** 카운트

## BR-2. 회원권(멤버십) 규칙

### BR-2.1 종류 및 금액

| 종류 | 기간(일) | 금액(원) |
|------|----------|----------|
| 1개월 | 30 | 100,000 |
| 3개월 | 90 | 270,000 |
| 6개월 | 180 | 480,000 |
| 12개월 | 365 | 840,000 |

### BR-2.2 만료일 계산
- 기본: `start_date + duration_days`
- **동결 기간은 만료일에서 제외** (동결 일수만큼 만료일 연장)
- 예: 1월 1일 시작, 30일 회원권, 1/10~1/15 동결(6일) → 만료일 2월 5일

### BR-2.3 동결(Freeze) 규칙
- 연간 최대 **30일** 동결 가능
- 1회 동결 시 최소 3일, 최대 15일
- 동결 중 시설 이용 불가
- 동결 기간은 `freeze_periods` 테이블에 기록

### BR-2.4 환불 규칙
- **가입 후 7일 이내:** 전액 환불
- **7일 초과:** 잔여 일수 비례 환불, 위약금 10% 적용
  - `환불 금액 = (잔여 일수 / 전체 일수) x 총 금액 x 0.9`
- 동결 기간은 이용 일수에서 제외
- PT 패키지 환불: `잔여 횟수 x 단가 x 0.9` (무료 체험은 잔여 횟수에 미포함)

## BR-3. PT 세션 규칙

### BR-3.1 패키지
| 패키지 | 횟수 | 금액(원) |
|--------|------|----------|
| 기본 | 10회 | 500,000 |
| 표준 | 20회 | 900,000 |
| 프리미엄 | 30회 | 1,200,000 |

### BR-3.2 무료 체험 세션
- 신규 회원 대상 **최대 3회** 무료 체험 제공
- 무료 체험은 `is_trial = True`로 구분
- **무료 체험 횟수는 유료 잔여 횟수에 포함하지 않음**
- 무료 체험 완료 후 유료 전환율 통계에 활용

### BR-3.3 세션 상태
| 상태 | 설명 |
|------|------|
| scheduled | 예약됨 |
| completed | 완료 |
| cancelled | 취소 (24시간 전까지) |
| no_show | 무단 불참 (횟수 차감) |

- `no_show`는 사용 횟수에 포함 (패널티)
- `cancelled`은 사용 횟수에 미포함 (복원)

## BR-4. 트레이너 매칭 규칙

### BR-4.1 전문 분야
- 트레이너는 1~3개의 전문 분야를 보유
- 전문 분야: 체중감량, 근력강화, 재활, 체형교정, 스포츠
- **매칭 시 전문 분야가 정확히 일치해야 함** (부분 매칭 불가)
  - "체중" 검색 → "체중감량" 매칭 안 됨
  - "체중감량" 검색 → "체중감량" 매칭됨

### BR-4.2 가용량
- 트레이너별 최대 담당 회원 수: `max_clients`
- 현재 담당: `current_clients`
- **가용량(current_clients < max_clients)이 있는 트레이너만 추천**
- 동일 조건 시 가용량이 많은 트레이너 우선

## BR-5. 통계 규칙

### BR-5.1 시간대 기준
- 모든 날짜/시간은 **KST(Asia/Seoul, UTC+9)** 기준으로 집계
- DB 저장은 UTC, 조회 시 KST 변환
- 예: 2026-01-31 23:30 KST 가입 → 1월 회원으로 집계

### BR-5.2 센터별 회원 수
- **활성 회원만** 카운트 (status = 'active')
- 이관 회원은 **현재 소속 센터**에서만 카운트
- `transfer_history`의 `transferred_at` 이후에는 새 센터 소속

### BR-5.3 잔여 일수/횟수 계산
- 회원권 잔여 일수: `만료일(동결 반영) - 오늘`
- PT 잔여 횟수: `패키지 총 횟수 - 사용 횟수`
- 사용 횟수: `completed` + `no_show` (cancelled 제외)
- **무료 체험은 잔여 횟수 계산에서 제외**

## BR-6. 챗봇 규칙

### BR-6.1 지원 질문 유형
| 키워드 | 응답 내용 |
|--------|-----------|
| 잔여, 남은 | 회원권 잔여 일수 또는 PT 잔여 횟수 |
| 만료, 언제까지 | 회원권 만료일 |
| 동결, 정지 | 동결 신청 방법 및 규칙 안내 |
| 환불 | 환불 규칙 안내 |
| 영업시간, 운영시간 | 센터 영업시간 안내 |
| 위치, 주소 | 센터 위치 안내 |
| PT, 트레이너 | PT 관련 안내 |

### BR-6.2 매칭 규칙
- **유의어를 포함하여 매칭**해야 함
  - "잔여" = "남은" = "남아있는"
  - "만료" = "언제까지" = "유효기간"
  - "동결" = "정지" = "일시중단"
- 매칭되지 않는 질문: "담당자에게 연결해 드릴게요" 응답
